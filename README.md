# Uncode IDE

Uncode 是一个面向云研发时代设计的下一代概念性 IDE。特性：

 - 万物代码化。DSL 设计
 - 全自动化工作流。DevOps as code
 - 填空式编程。（想想还是很美好的，大部分人只是搬砖的
 - 选择题式编程。AI 代码搜索（为项目加分
 - 时尚的 UI。（这个肯定没有
 - 云研发架构 water 适配。（暂时不支持多语言
